# System Evaluation Checklist

## 1. Execution
- Does the system run with a single command?
- Are failure modes reproducible?

## 2. Consistency
- Are invariants validated automatically?
- Is journal replay deterministic?

## 3. Resilience
- Can the system recover from full failure?
- Are partial states handled correctly?

## 4. Observability
- Are metrics and health visible?
- Can system state be interpreted quickly?

## 5. Architecture clarity
- Are layers clearly separated?
- Is control flow understandable without code reading?
